# Sobre o projeto

Um projeto criando em uma versão para web, pode ser usado localmente também usando um simulador de servidor web exemplo o xampp, porém super leve porque usa como banco o sqlite3.

Ele é bem minimalista porque faz somente o necessário  que é gerenciar suas tarefas do dia a dia, pode ser usado por mais de uma pessoa porque cada um tem seu usuário que consegue criar, editar, apagar ou marcar como concluída suas tarefas.

Um projeto feito para mostrar todas as minhas técnicas com a linguagem php, javascript, html, css e manipulação de banco de dados.

# Tecnologias utilizadas

### Sobre desenvolvimento
Foi usado uma estrutura MVC diferente das convencionais porque usei javascript e jquery para fazer as requisições e recuperar os dados e as views do app, usado o sqlite 3 porque é um banco de dados leve e de distribuição livre. **tempo para concluir esse projeto foi de 48 horas**


### Back end
- Jquery
- JavaScript
- PHP 7.4

### Front end
- HTML5
- CSS3
- BOOTSTRAP 4.4

# Como executar o projeto

**Pré-requisitos:**. 
- Sistema Operacional Linux, Windows ou MAC
- XAMPP instalado ou qualquer outro para simular servidor web (Somente necessário o php para funcionar)
- Navegador de internet.

```bash
# clonar repositório
git clone https://github.com/fabianobasso/todo_list.git
```
- Copiar a pasta do projeto para pasta publica do XAMPP
- Abrir o projeto no navegador.
